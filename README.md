# employee-salary-manager
Aplicação em Java para gerenciamento de funcionários, com validação de ID único e aumento de salário por porcentagem utilizando encapsulamento e ArrayList.

## Funcionalidades

- Cadastro de funcionários (id, nome e salário)
- Validação de ID único
- Busca de funcionário por ID
- Aumento de salário por porcentagem
- Listagem atualizada de funcionários

## Tecnologias

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
